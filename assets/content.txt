CSS Framework adalah kumpulan kode CSS dan JavaScript yang sudah jadi dan siap digunakan untuk mempercepat proses pembuatan tampilan website. Framework ini mengandung berbagai komponen seperti layout, warna, jenis huruf, tombol, formulir, tabel, dan banyak lagi yang dapat digunakan untuk membuat tampilan website yang responsif dan menarik tanpa perlu membuat semuanya dari awal.
